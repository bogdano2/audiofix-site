AudioFix — Windows Audio Device Manager
========================================

Thank you for your purchase!

QUICK START
-----------
1. Right-click AudioFix.ps1
2. Select "Run with PowerShell"
3. Follow the on-screen prompts

The script will:
  - Identify all your audio devices
  - Play a test tone through each one so you can confirm which is which
  - Lock your preferred defaults so Windows stops switching on its own

REQUIREMENTS
------------
- Windows 10 or Windows 11
- PowerShell 5.1+ (built into Windows)
- Administrator access (the script will prompt for elevation)
