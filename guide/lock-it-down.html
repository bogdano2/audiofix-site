Lock it down

You've identified your real devices. Now disable everything else so Windows can't silently switch to the wrong one.

This is the most important step

Disabling unused devices is the single most effective thing you can do to prevent audio problems.

Remember from the first guide: the more active devices Windows sees, the more likely it is that the default will silently change. By reducing the list to only what you actually use, you eliminate the opportunities for drift.

Think of it like TV remotes on a couch — the more there are, the more likely someone grabs the wrong one. Remove the ones you never use.

Set your tested device as default first

Before disabling anything, make sure the device you tested is set as the system default.

Steps

  1. Open Sound Settings: press Win + R, type ms-settings:sound, press Enter
  2. In the Output section, select your tested device from the dropdown
  3. Click the device name to open its properties
  4. Under "Set as default sound device," ensure it says "Used as default"
  5. Repeat for the Input section with your tested microphone
Three audio roles. Windows has three: Console, Multimedia, and Communications. The Settings UI usually sets Console and Multimedia together, but Communications may need to be set separately. Check all three if available.

This is why Zoom might work but Spotify doesn't, or vice versa. If different roles point to different devices, some apps will use the "wrong" one. Setting all three roles to the same device ensures consistency.

Open the full device list

The regular Sound page only shows active devices. To disable devices, you need the full list.

Press Win + R, type ms-settings:sound-devices, press Enter.

This opens "All sound devices" — every output and input device, including disabled ones.

Disable output devices you don't use

For each output device that is NOT the one you tested and confirmed:

  1. Click the device name
  2. On its properties page, click "Don't allow" (or "Disable")
  3. Go back and repeat for each unwanted output device

Disable input devices you don't use

Same process, now for microphones. Common devices to disable:

  • Stereo Mix — this is not a microphone. It captures whatever plays through your speakers. Disable it unless you specifically need to record system audio.
  • Webcam microphone — if you use a dedicated USB mic, disable the webcam's built-in mic to prevent accidental switching.
  • Any input device that showed 0% in your mic test — it's not picking up your voice.
Stereo Mix as your default mic is one of the most common causes of "nobody can hear me on calls." If this was your default, disabling it and setting your real mic as default will likely fix the problem immediately.

Verify your cleanup

Go back to the main Sound page (ms-settings:sound) and check:

  • Output dropdown shows only the device(s) you actually use
  • Input dropdown shows only your actual microphone
  • Your chosen output is marked as default
  • Your chosen input is marked as default
  • Volume sliders are at a reasonable level (not zero)

That's it for the OS layer. Your defaults are set and the noise is gone.

What if it breaks again?

If audio stops working after a Windows update, driver change, or plugging in new hardware, come back to this page. The process is the same every time: check defaults, test, disable what shouldn't be there.
