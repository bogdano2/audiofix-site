Audio device not showing up on Windows 11?

Windows macOS coming soon Linux coming soon

If a wired or USB audio device isn't showing up in Sound Settings, the most common cause is the device being disabled in Device Manager — not a missing driver. Windows has built-in driver support for the vast majority of audio hardware, so this is rarer than you'd think.

Check Device Manager for disabled devices

Sound Settings only shows devices that Windows considers active and available. Device Manager goes deeper — it tracks every piece of hardware Windows knows about, including devices that are disabled, disconnected, or broken. If your audio device vanished from Sound Settings, this is where you'll find it.

  1. Press Win+R, type devmgmt.msc, press Enter. This opens Device Manager directly.
  2. Expand the Sound, video and game controllers category by clicking the arrow next to it.
  3. Look for your audio device in the list. It may have a down arrow ↓ (meaning it's disabled) or a yellow triangle ⚠ (meaning there's a driver error).
  4. Right-click the disabled device and select Enable device.
  5. Open Sound Settings again (Win+R, type ms-settings:sound, press Enter) — the device should now appear in the output or input list.
This fixes the problem the majority of the time. A disabled device is invisible to Sound Settings but still known to Windows. Enabling it brings it right back.
Don't see "Sound, video and game controllers" at all? Click View → Show hidden devices in the Device Manager menu bar. The entire category may be hidden because every device in it is currently disabled.

Show hidden devices

Windows hides devices that have been disconnected or disabled. They're still registered in the system — Windows remembers they existed — but they won't appear in Device Manager's default view. This is a common reason people think a device has completely vanished: it's there, just invisible.

  1. In Device Manager, click View in the menu bar, then select Show hidden devices.
  2. Expand Sound, video and game controllers again. You may see new entries that weren't visible before.
  3. Grayed-out or ghosted entries are hidden devices — they're installed but not currently active.
  4. Right-click any grayed-out audio device you recognize and select Enable device.
Device appears but won't enable? If your device shows up grayed out but refuses to enable, the driver may be corrupted. The next section covers how to fix that.

Update the driver from Device Manager

When you click "Update driver" in Device Manager, Windows checks two places: its own local driver store (a cache of drivers that shipped with Windows or arrived via Windows Update) and Microsoft's online driver catalog. It does not search third-party websites — this is a safe, built-in process.

  1. In Device Manager, right-click your audio device (even if it has a yellow triangle or error icon).
  2. Select Update driver.
  3. Click Search automatically for drivers.
  4. Wait for Windows to search. It will either install a newer driver or tell you that the best available driver is already installed.

If Windows says the best driver is already installed but the device still isn't working, try a clean reinstall: right-click the device → Uninstall device → check the box labeled "Attempt to remove the driver for this device" → click Uninstall. Then, in the Device Manager menu bar, click Action → Scan for hardware changes. Windows will rediscover the hardware and install a fresh copy of the driver from scratch.

This approach works when Windows has the correct driver but it became corrupted — perhaps by a failed update, a power interruption during install, or a conflict with another driver. The reinstall gives Windows a clean slate. If the driver simply isn't in Windows Update at all, you'll need to get it from the manufacturer directly.

Find the driver from the manufacturer

If Device Manager can't find a working driver through Windows Update, you'll need to download it directly from the company that made the device. This is the only reliable source for drivers that Windows doesn't already have.

How to identify your device

  • Check the physical device itself for a manufacturer name and model number — it's usually printed on the bottom or back.
  • Check the device's original packaging or manual if you still have it.
  • In Device Manager, right-click the unknown or broken device → Properties → Details tab → select Hardware Ids from the Property dropdown. This shows the vendor and product ID (e.g., USB\VID_1235&PID_8211) that you can search for online to identify the manufacturer.

Where to download drivers

Manufacturer Where to find drivers
Realtek Realtek's official download center
Creative (Sound Blaster) Creative's support and downloads page
Your laptop brand (Dell, HP, Lenovo, ASUS) The manufacturer's support page — enter your model number
Intel (display/HDMI audio) Intel's driver support assistant

Never download drivers from third-party "driver update" sites

Sites like "DriverBooster," "DriverEasy," "DriverPack Solution," and similar tools are not trustworthy sources for drivers. Here's why:

  • They frequently bundle adware, potentially unwanted programs (PUPs), or outright malware with their downloads.
  • They may install incorrect or outdated drivers that cause more problems than they solve.
  • Their "free scans" are designed to alarm you into paying for software you don't need.

The only safe sources are: the device manufacturer's website, your PC manufacturer's support page, and Windows Update.

If a website asks you to install a "driver scanner" or "driver updater tool," close the tab.

When to get professional help

If you've checked Device Manager, shown hidden devices, tried updating and reinstalling the driver, and downloaded from the manufacturer — and the device still doesn't appear — the issue is likely hardware-level. You've done real diagnostic work to get here, and that matters.

Possible hardware causes

  • A dead audio jack or USB port — try a different port on your computer
  • A damaged cable or connector — try a different cable if possible
  • A failed onboard audio chip, especially on older motherboards
  • A front panel audio header that isn't connected to the motherboard — common in custom-built PCs
At this point, a professional technician can diagnose whether it's a software or hardware problem. You've already ruled out the most common software causes — that's valuable diagnostic information to share with them.

Once the device appears

If you got the device to show up — whether by enabling it in Device Manager, updating the driver, or downloading from the manufacturer — head back to the guide to make sure it's properly configured.

Devices that required manual intervention are especially prone to default-switching issues. Windows may not prioritize them correctly, or they may disappear again after an update. Setting the device as your default and disabling unused devices prevents this from happening again.
