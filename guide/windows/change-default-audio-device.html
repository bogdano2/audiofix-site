How to change the default audio device on Windows 11

Windows macOS Linux

To change your default speakers or microphone on Windows 11, open Sound Settings (Win+R, type ms-settings:sound, Enter). Select the device you want from the Output dropdown for speakers, or the Input dropdown for microphone. The change takes effect immediately.

Change default output device (speakers/headphones)

The output device is where Windows sends all sound — music, system notifications, game audio, video playback. When you change the default output device, every app that hasn't overridden the system default will immediately switch to the new device.

Method A: Sound Settings dropdown (fastest)

  1. Press Win+R, type ms-settings:sound, and press Enter.
  2. Under Output, find the dropdown labeled "Choose where to play sound."
  3. Click the dropdown and select your preferred speakers or headphones.
  4. Sound switches immediately — no restart or confirmation needed.
Quick alternative: Click the speaker icon in the system tray (bottom-right of taskbar), then click the arrow () next to the volume slider. This opens a device picker where you can switch output devices in two clicks.

Method B: Classic Sound panel (more control)

  1. In Sound Settings, scroll to the bottom and click "More sound settings."
  2. In the classic Sound panel, find the Playback tab.
  3. Right-click your preferred device and select "Set as Default Device."
  4. A green checkmark appears next to the new default.

Change default input device (microphone)

The input device is the microphone Windows uses for voice calls, recording, and dictation. Changing it follows the same pattern as the output device.

Method A: Sound Settings dropdown

  1. Press Win+R, type ms-settings:sound, and press Enter.
  2. Scroll down to the Input section.
  3. Click the dropdown labeled "Choose a device for speaking or recording" and select your preferred microphone.

Method B: Classic Sound panel

  1. Open More sound settings (bottom of Sound Settings page).
  2. Switch to the Recording tab.
  3. Right-click your preferred microphone and select "Set as Default Device."
Watch out for "Stereo Mix." If you see a device called Stereo Mix in the Recording tab, don't set it as your default microphone. Stereo Mix captures your computer's output audio (what you hear), not your voice. It's useful for specific recording scenarios, but if it becomes your default input, callers on Zoom and Teams will hear your desktop audio instead of your voice.

Set the Communication Device

Windows actually tracks three separate device roles, not just one. Understanding these roles explains why your call audio sometimes goes to a different device than your music.

The three audio device roles in Windows and what each one controls
Role What uses it How to set it
Default Device General playback — music, videos, games, system sounds Right-click → Set as Default Device
Default Communication Device Voice/video call apps — Zoom, Teams, Discord, Skype Right-click → Set as Default Communication Device
Default Recording Device Microphone input for calls, dictation, and recording apps Recording tab → right-click → Set as Default Device

How to set the Communication Device

  1. Press Win+R, type ms-settings:sound, press Enter.
  2. Scroll to the bottom and click "More sound settings" to open the classic Sound panel.
  3. On the Playback tab, right-click the device you want to use for calls.
  4. Select "Set as Default Communication Device."
  5. The device will show a green phone icon instead of (or alongside) the green checkmark.
Why does this matter? If your Default Device is your desktop speakers but your Communication Device is your USB headset, music plays through your speakers while Zoom calls go through your headset. This is actually useful if you set it up intentionally — but confusing if Windows did it without telling you.

Keyboard shortcut method

You can navigate Sound Settings entirely from the keyboard. This is faster once you know the shortcuts, and essential if your mouse isn't working or you prefer keyboard navigation.

Keyboard shortcuts for navigating Sound Settings on Windows 11
Shortcut Action
Win+R, then type ms-settings:sound Open Sound Settings directly
Win+R, then type mmsys.cpl Open classic Sound panel directly (skips the Settings app)
Tab / Shift+Tab Move between controls in Sound Settings
Space or Enter Open a dropdown or activate a button
Arrow Up / Arrow Down Cycle through devices in a dropdown list
Win+Ctrl+V Open Volume Mixer (per-app volume control)
Power-user tip: Win+R → mmsys.cpl opens the classic Sound panel directly, which gives you access to both the Default Device and Default Communication Device options in one place. The modern Settings app only lets you change the Default Device.

Why does Windows keep changing it?

If you've changed your default device only to find Windows switched it back later, you're not imagining things. Windows automatically reassigns the default audio device when any of these things happen:

  • A new device connects. Plugging in an HDMI cable, pairing Bluetooth headphones, or connecting a USB headset can cause Windows to switch the default to the newly detected device.
  • Windows Update resets preferences. Major feature updates (and occasionally cumulative updates) can reset audio device assignments.
  • An app overrides the system default. Apps like Zoom, Teams, Discord, and many games have their own audio device settings. Even if the system default is correct, the app may be sending audio somewhere else.
  • A device disconnects and reconnects. Bluetooth devices that lose and regain connection, or USB devices that get unplugged and re-plugged, can trigger a default reassignment.

For the full explanation of why this happens at the OS level, see Why it happens.

Make it permanent

Changing the default device is the first step, but if Windows keeps switching it back, you need to remove the devices that cause the switches. The most effective method is to disable every audio device you don't use.

Think of your audio devices like TV remotes on a couch — the more there are, the more likely someone grabs the wrong one. Fewer active devices means fewer opportunities for Windows to silently switch your default.

  1. Open the classic Sound panel: Win+R, type mmsys.cpl, press Enter.
  2. Right-click any device you don't use and select "Disable."
  3. Repeat for both the Playback and Recording tabs.
  4. Common candidates: Stereo Mix, HDMI outputs you never use, virtual audio cables from uninstalled apps.
Don't worry about breaking anything. Disabling a device doesn't uninstall it. You can always right-click in the Sound panel, select "Show Disabled Devices," and re-enable anything you need later.
