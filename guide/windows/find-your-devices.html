Find your devices

Windows macOS Soon Linux Soon

Before you fix anything, you need to see what your OS sees. Windows might be managing a dozen audio devices you didn't know existed.

Open Sound Settings in 3 seconds

The fastest way to get there. Three keystrokes, no clicking through menus.

Win +plus R type ms-settings:sound Enter
Bookmark this URI. You'll use it often.

URI Opens
ms-settings:sound Sound main page
ms-settings:sound-defaultoutputproperties Default output properties
ms-settings:sound-defaultinputproperties Default input properties
ms-settings:sound-devices All devices (enable/disable)

Find your output devices

The Output section at the top of the Sound page shows which device Windows is currently sending sound to. Click the dropdown to see all available output devices.

Choose where to play sound:

Speakers (Realtek(R) Audio) DEFAULT
LG TV (Intel(R) Display Audio)
Digital Audio (S/PDIF)
Speakers (USB PnP Audio Device)

The one marked DEFAULT is where your sound is going right now.

Find your input devices

Scroll down to the Input section. This shows which microphone Windows is currently listening to.

Choose a device for speaking or recording:

Microphone (Realtek(R) Audio) DEFAULT
Microphone (USB PnP Audio Device)
Stereo Mix (Realtek(R) Audio)
Stereo Mix is not a microphone. It captures whatever plays through your speakers. If this is your default, that's probably your problem.

Spot the impostors

Not everything in the list is a real audio device you'd want to use. Here's a quick cheat sheet.

Device name pattern What it actually is Usually want it?
Speakers (Realtek) Built-in laptop/motherboard speakers Often
Headphones (Realtek) The 3.5mm headphone jack If wired
LG TV / Display Audio HDMI audio through your monitor Rarely
Digital Audio (S/PDIF) Optical audio output Rarely
USB PnP Audio Device Docking station, webcam, or USB hub with audio Depends
Stereo Mix Loopback — records system audio Almost never
Bluetooth headset Wireless headphones/earbuds When connected

Don't see your device at all?

If your speakers, headphones, or microphone aren't showing up in Sound Settings, the fix depends on the connection type:

Bluetooth / wireless — pair and connect the device first Wired / USB — check Device Manager and drivers

Check what's set as default

The device with the checkmark or "DEFAULT" label is where Windows routes all sound (output) or listens (input). If this is wrong, everything downstream is wrong.

Look at your Sound settings right now. Is the default output device the one connected to your actual speakers? Is the default input device your actual microphone?

If the defaults look right and you still have problems, the issue might be volume, a mute switch, or an app override. We'll cover testing next.

Quick volume and mute check

While you're here, verify the basics.

  • Output volume slider is above zero
  • Output device is not muted (check the speaker icon)
  • Input volume/sensitivity slider is above zero
  • Microphone is not muted (check software AND any hardware mute button)
  • Physical device is actually powered on

These account for more "broken audio" reports than you'd expect.
