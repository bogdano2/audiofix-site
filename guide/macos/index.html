Fix Audio on macOS

We're building a guide for fixing audio device switching on macOS using CoreAudio and Audio MIDI Setup. The same four-phase approach applies: identify, set default, disable unused, check apps.

What to expect

The macOS guide will walk you through fixing audio device problems using the tools Apple provides. Here's what it will cover:

  • CoreAudio API — how macOS manages audio endpoints under the hood, why defaults drift, and how to take control
  • Aggregate devices — combining multiple audio interfaces into one virtual device for consistent routing
  • Per-app audio routing via Audio MIDI Setup — configuring individual applications to use specific input and output devices
In the meantime: The concepts in our guide to why audio breaks apply to every operating system. Understanding why your OS switches devices is the first step to fixing it, regardless of platform.
